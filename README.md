## 項目展示

[Demo](https://random-quote-machine-ecru.vercel.app/)（2022.04.19 完成）

※ 顏色可能過亮，觀看時請注意！



## 目的

練習 React、TS，並結合 TailWind CSS 排版



## 功能說明

點下按鈕「New quote」即可隨機變更整體顏色，並產生一段名言及該名人。

左方 Twitter、Tumblr 按鈕，按下後可跳轉至發文。
